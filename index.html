1. #id - Ідентифікатор визначає унікальне ім'я елемента, яке використовується для зміни його стилю і звернення до нього через скрипти.
  2. .class - Класи застосовують, коли необхідно визначити стиль для одного або декількох елементів веб-сторінки.
  3. * - Універсальний селектор дозволяє прописати стилі для всіх елементів.
  4. Селектори типу вибирають елементи HTML-документа за їхнім тегом. Наприклад, селектор p вибере всі насторінці.
  5. Вкладені селектори - Встановити стиль для b тільки коли він розташовується всередині контейнера p. Таким чином можна одночасно встановити стиль для окремого елемента.
  6. .class1.class2 - До будь-якого елемента одночасно можна додати кілька класів.
